# 🧠 Dynamic Wumpus Logic Agent (Web App)

## 📌 Project Title

Dynamic Wumpus Logic Agent using Propositional Logic and Resolution

## 🎯 Objective

This project implements a dynamic pathfinding agent based on the Wumpus World environment.
The agent uses propositional logic and resolution refutation to deduce safe cells and avoid hazards.

---

## ⚙️ Features

✔ Dynamic Grid Size (User-defined rows & columns)
✔ Random Pit and Wumpus Placement
✔ Breeze and Stench Percepts
✔ Knowledge Base (Propositional Logic)
✔ Resolution-Based Inference Engine
✔ Real-Time Grid Visualization
✔ Metrics Dashboard (Inference Steps + Percepts)

---

## 🧠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Canvas API
* Propositional Logic
* Resolution Algorithm
